Программа сетевой чат MyMessenger:

Программа разделена на 2 блока: Сервер и клиент.
Связь организована на сокетах.
В программе использовались следующие технологии:
Потоки ввода вывода по сети.
Работа с файлами создание, чтение/запись.
Базы данных: построена на СУБД SQLite.
Управление БД язык SQL. 
Сервер построен на технологии многопоточности с использованием управляемого массива потоков фонового обмена потоками через  ExecutorService
Для графического интерфейса использовалась библиотека JavaFX.
Для контроля работы сервера используется система логирования Log4j.
Для сборки сервера используется система сборки Maven.
