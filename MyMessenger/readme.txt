Запуск и настройка подключений.
1 Устанавливаем и запускаем сервер.
2 На этом же компьютере где и сервер запускаем Клиента
3 На клиенте входим логин: admin, пароль: admin
4 На клиенте делаем команду /server
5 Копируем клиента с сервера на удаленный компьютер либо файл connect.sys
6 Справочник команд /info


При первом запуске на сервер создастся файл connect.sys где:
PORT:50105   - порт 
COUNTCLIENTS:3 - максимальное количество клиентов
IPSERVER:192.168.1.102 - ip адрес сервера (указан будет ip вашей машины в локальной сети)
